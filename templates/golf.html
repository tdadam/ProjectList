Golf Score Card

This was the first project to use javascript, which causes action rather than just structure/format.

Features:

  • Retrieves information from two sites, Swing By Swing and Google Maps
  • Works for any golf course in the Swing By Swing database (with the ID code)
  • Map opens to the golf course clubhouse on page load
  • Yardage will display based on player selection (Men/Women/Champion/Pro Tee)
  • Clicking on a hole number, yardage, par, or handicap (if available) will recenter map
  • Tee and hole indicated on map, as well as path from last hole and to next tee
  • Total and over/under calculated as each hole total entered
  • Options only show and/or function if applicable or available
  • When round is finished, message opens declaring winner and gives an evaluation based on over/under
  • Direct url: tdadam.github.io/GolfCard
