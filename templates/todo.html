To Do List

This appears to be a more simple project, but was the first using the Angular framework for javascript. It has become my sandbox for playing with Angular features and will be receiving updates as I attempt new Angular features. The purpose of Angular is to have a single page website, while changing the information displayed and having data persist when changing tabs/views.

Features:

  • Tasks and About tabs display different information without reloading the page
  • Lists and items are not deleted when switching between Tasks and About
  • Items are sorted by date due
  • Tasks can be deleted or marked completed and archived
  • Archived tasks can be removed or recovered
  • Future Goal: save the lists and items to the local computer storage so reloading does not clear the information (not yet implemented)
  • Direct url: tdadam.github.io/ngToDo
